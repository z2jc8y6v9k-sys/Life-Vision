# My Life Vision v62 — Goal Lifecycle + Achievements

Changes:
- Marking a project **Completed** now sets progress to 100% and removes it from the main Workbook.
- Completed goals are removed from Priority Stack, Inventory Goals, and Workplan.
- Adds a new **Achievements** section under Dashboard.
- Achievements preserves the goal title, category, why, wins/reflection, and lessons.
- When completing a goal, the app optionally asks: “What did accomplishing this make possible?” and saves the answer with Wins.
- Completed goals can be returned to the Workbook or archived from Achievements.
- Archived goals stay out of the Workbook and Achievements.
- No SQL changes required; this uses the existing status, progress, wins, and lessons fields.
