# My Life Vision v33

A simple GitHub-ready personal life vision web app.

## Files

- `index.html` — main app structure
- `styles.css` — visual design and responsive layout
- `script.js` — editable workbook fields and local browser storage

## How to use

1. Create a new GitHub repository.
2. Upload these files to the root of the repo.
3. Turn on GitHub Pages: Settings → Pages → Deploy from branch → `main` → `/root`.
4. Open the GitHub Pages URL.

## Notes

- Workbook text is saved in the browser using `localStorage`.
- The Dashboard links jump each section to the top of the screen.
- The floating `↑ Top` button returns to the top from anywhere.
