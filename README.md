# My Life Vision App v14 Bullets

This is the v14 pre-launch version with bullet-friendly written sections.

Adds:
- Bullet button on:
  - Key Results
  - People
  - Resource Notes
  - Today / This Week
  - Next 30 Days
  - Next 12 Months
- Pressing Enter after a bullet automatically starts the next bullet.
- Existing Supabase data is preserved.

Still includes v14:
- Priority Stack
- Life Seasons
- Resource Profile:
  - Time Required
  - Money Required
  - Physical Demand

Important:
Run `supabase_v14_migration.sql` once in Supabase if you have not already run it.

Deploy:
1. Run v14 SQL migration if not already done.
2. Upload/replace:
   - index.html
   - styles.css
   - app.js
   - README.md
3. Do not upload the SQL file unless you want it as documentation.
